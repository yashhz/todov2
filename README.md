# Planify

A Life Operating System designed to bridge the gap between daily "busy-work" and long-term vision using a powerful Interlinking Engine.

Planify is not just a to-do list; it is a "Premium Digital Glass" workspace built for speed and focus, helping you drive tangible progress toward what really matters.

## Documentation
- [Architecture & Overview](APP_OVERVIEW.md)
- [Design Philosophy](DESIGN_PHILOSOPHY.md)

## Tech Stack
- Frontend: React + TypeScript + Vite
- State Management: React Context API + LocalStorage
- Styling: Pure CSS Modules + Custom Tokens (`src/styles/tokens.css`)
- Icons: `lucide-react`

## Commands
- `npm install`: Install dependencies
- `npm run dev`: Start local development server
- `npm run build`: Compile and build production-ready files
